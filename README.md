# unchain.js — minimal SPA helper

A lightweight, zero-dependency core for building and experimenting with Single Page Applications.

> ⚠️ Not for production use.  
> Intended for small apps, rapid prototyping, and learning.

Experimental — under development

## ⚡ Why unchain?

- **Zero dependencies:** No `node_modules`.
- **Fast prototyping:** Build SPAs quickly without build steps or extra complexity.

## 🎯 When to use

- **Local tools:** Internal dashboards or small apps.
- **Prototyping:** Test UI/UX before choosing a framework.
- **Experiments:** Vanilla JS sandboxes.
- **Learning:** Understand SPA fundamentals.

## ⚙️ Core features

- Basic SPA routing (hash-based)
- State management helpers
- Rendering and reactivity
- Event handling
